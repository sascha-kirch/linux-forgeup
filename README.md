# linux-forgeup
Linux System Tool to automate setup and configuration of my debian based linux distros

# Installion

1. Clone this repository:
```bash
git clone https://github.com/sascha-kirch/linux-forgeup.git
```

2. Run the setup script:
```bash
./run.sh
```
